# data-science-notes
This repository is where I collect small experiments, intuition-building notebooks, a machine learning "curiosity corner”, little examples of OSINT work, etc.

It's very much under construction and intentionally informal — a place for interesting ideas and small demonstrations rather than polished portfolio projects.

In the ML curiousity corner you'll find:

**[Meditations on Modelling](https://github.com/greeny-blue/data-science-notes/tree/main/ml-curiosity-corner/meditations-on-modelling)** - short (and possibly thought-provoking) reflections on machine learning and artificial intelligence, our use of them and their limits.
  - [#1 All Models Are Wrong, But Some Are Useful](https://github.com/greeny-blue/data-science-notes/blob/main/ml-curiosity-corner/meditations-on-modelling/Meditation-001-All_models_are_wrong.ipynb)
  - [#2 Machine Leaning or "Machine Rummaging"?](https://github.com/greeny-blue/data-science-notes/blob/main/ml-curiosity-corner/meditations-on-modelling/Meditation-002-Machine_rummaging.ipynb)
  - [#3 Models Produce Answers, Not Understanding](https://github.com/greeny-blue/data-science-notes/blob/main/ml-curiosity-corner/meditations-on-modelling/Meditation-003-Models_produce_answers_imgfix.ipynb)

**Extreme Examples** - short, extreme example experiments that explore and highlight interesting behaviour in machine learning systems (that everyone doing ML should be aware of!).
